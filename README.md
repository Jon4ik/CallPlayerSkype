CallPlayerSkype - плагин позволяющий администратору сервера запросить Skype игрока через админку

# Требования:
* SourceMod 1.5.2 и выше
* SourceBans (не обязательно)

# Доступные команды:
* !testsb - проверка работы библиотеки Sourcebans
